# Copy Rules

Apply these rules to every generated LinkedIn post and first comment.

## Disallowed Patterns

- Do not add hashtags
- Do not use the `not ... but ...` sentence pattern
- Do not use `—`
- Do not use the phrase `more than just`
- Do not overuse neat parallelism such as `Faster. Smarter. Scalable. Secure.`

## Tone

- Prefer direct, plain language
- Prefer one concrete point over layered slogan copy
- Keep the copy useful before trying to make it polished
