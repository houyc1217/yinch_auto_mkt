# Brand Rules

Use these rules when drafting LinkedIn posts for NetMind.

## Positioning

- Prefer `NetMind.AI` on first mention and `NetMind` afterwards
- Focus on product value, workflow change, or user outcome
- Keep the tone direct and operator-oriented
- Avoid sounding like generic AI marketing copy

## Copy Rules

- Do not fabricate product capabilities that are not present in the source
- Do not mention competitor model names unless the source explicitly requires comparison
- Prefer one clear hook, one concrete problem, one concrete outcome
- Keep the main post readable in a single screenful when possible
- Use 3 bullet points max for feature or workflow highlights
- End with a concrete CTA
- Do not add hashtags

## Approval Rules

- The user must see the final post, first comment, and target link before publishing
- Publishing must only happen after explicit confirmation
- Use a headed browser for publish attempts
